Robotron

O Robotron 2000 é um projeto prático que foi desenvolvido durante o curso "JavaScript: manipulando o DOM".

Utilizamos o JavaScript para manipular o DOM (Document Object Model) da página web. Isso significa que aprendemos a interagir com os elementos HTML e CSS da página, adicionando funcionalidades e interatividade ao projeto.

Algumas das coisas que aprendemos:

-Como selecionar e manipular elementos HTML usando JavaScript;
-Como criar funções e lógica de programação para controlar o robô;
-Como realizar cálculos e operações com os dados do robô;
-Como atualizar a interface do usuário de acordo com as ações do robô;

Número Secreto 1

O foco principal era aplicar os conceitos básicos de lógica de programação, como:

-Variáveis: Aprendi armazenar informações, como o número secreto gerado pelo computador, em variáveis.

-Condicionais (if-else): Utilizei condicionais para verificar se o palpite do usuário é maior, menor ou igual ao número secreto.

-Loops: Implementei loops para permitir que o usuário continue tentando adivinhar o número até acertar.

-Boas práticas de programação: Como dar nomes significativos às variáveis e manter um código organizado e legível.

Com esses conceitos, criamos um jogo interativo de adivinhação do "Número Secreto 1.0", colocando em prática todos esses conceitos fundamentais de lógica de programação.

Número Secreto 2

O projeto do Número Secreto 2.0 é baseado no jogo do número secreto. O objetivo principal é ensinar como usar o JavaScript para interagir com a tela e aplicar os conceitos de lógica de programação.

Adicionamos novas funcionalidades e melhorias visuais ao jogo do número secreto. Ao final, o projeto será colocado no ar para que outras pessoas possam acessá-lo e jogar.

O projeto visa aplicar os conhecimentos de lógica de programação e JavaScript em um jogo interativo, com o objetivo de disponibilizá-lo online para outras pessoas. Ajudando a colocar em prática tudo o que aprenderem durante o curso.

Alura Midi

O Alura Midi é um instrumento musical interativo que será desenvolvido ao longo do curso. Ele consiste em uma interface digital onde o usuário poderá tocar diferentes sons ao clicar em botões ou pressionar teclas do teclado. O código base em HTML e CSS já está pronto, então o foco será no desenvolvimento do JavaScript.

-Manipulação do DOM: Aprendi a selecionar, modificar e interagir com os elementos HTML da página usando JavaScript.

-Eventos JavaScript: Trabalhei com diferentes tipos de eventos, como cliques e teclas pressionadas, para criar a interatividade do Alura Midi.

-Funções JavaScript: Aprendi a criar e utilizar funções para organizar e modularizar o seu código.

-Condicionais e Loops: Estudei as estruturas de controle, como if/else e loops, para implementar a lógica do instrumento musical.

-Variáveis e Tipos de Dados: Aprendi declarar e manipular variáveis, bem como os principais tipos de dados em JavaScript.

Serenatto
